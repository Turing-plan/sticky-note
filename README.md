# Sticky Notes — 极简任务清单

## 为什么开发这个
- 快速、轻量地记录和勾选待办，不被复杂功能打断
- 纯键盘驱动（输入、选择、完成、删除），减少操作摩擦，保持专注
- 始终置顶、无边框透明的视觉，像贴在桌面的便签
- 不依赖云账号，数据本地保存，隐私与可靠性更好
- 足够简单：输入 → 完成/删除 → 进度反馈，形成闭环

## 软件的使用
- 添加任务：按 `Ctrl+N` 打开添加窗口，输入后按 `Enter`（或点击“添加”）
- 选择任务：使用 `↑/↓` 在任务间移动
- 开始计时：选中任务后按 `Enter`（左侧圆圈变为橙色并显示进行中符号）
- 打勾完成：选中任务后按 `Space`（结算耗时并显示在任务右侧）
- 删除任务：选中任务后按 `Delete` 或 `Backspace`
- 全部删除：按 `Ctrl+Delete`（不论焦点位置均可触发）
- 取消选择：按 `Esc`；关闭添加窗口：按 `Esc`
- 全局快捷键：`Ctrl+M` 呼出/隐藏窗口，`Ctrl+Q` 退出应用（macOS 支持 `Cmd+M/Cmd+Q`）
- 进度反馈：底部进度条实时显示已完成比例
- 快捷键速记：`Enter`=开始计时，`Space`=完成，`Backspace/Delete`=删除，`Ctrl+N`=新建，`Ctrl+Delete`=清空（或 `Ctrl+Shift+Delete`）
- 自动保存：任务会自动保存为本地 JSON，下次打开自动恢复
